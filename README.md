This standalone template repository is deprecated.

_fate_ now ships built-in project templates from the main [`nkzw-tech/fate`](https://github.com/nkzw-tech/fate) repository. Create a new _fate_ app with Vite+ instead:

```bash
vp create @nkzw:fate fate-app
```

See the [_fate_ Getting Started guide](https://fate.technology/guide/getting-started#template) for the current template instructions.
